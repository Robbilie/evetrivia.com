How it works

Projects

All parts of the EVE Trivia project are open sourced and hosted on GitHub. Here is a list of all the projects and their current status. This also acts as a semi dashboard for them.

Backends

Project Name Description Build Status Coverage Status PyPi Status Docs
Thanatos A Python library for generating trivia questions from the EVE SDE.
Erebus A simple Flask application to take questions from Thanatos and server them via an HTTP API.
Nyx

Frontends

For now the only frontend is the one integrated into the play portion of this site. At a later date a standalone web frontend will be created for the project. The posability also exists of creating dedicated mobile frontends for EVE Trivia but at this time nothing is planned beyond the single web frontend.
